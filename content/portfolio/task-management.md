## Overview

社内管理システムの一部として、タスク管理モジュールのフロントエンド開発を担当。  
プロジェクトごとのタスク分配、工数記録、作業進捗の可視化機能を実装。

## Tech Stack

**Frontend:** React (担当部分)  
**Features:** タスク管理, タグシステム, タイマー機能

## Features

- 📝 **Task Assignment** - タスクの作成と担当者への割り当て
- 🏷️ **Tag System** - 柔軟なタグ付けによる分類管理
- ⏱️ **Time Tracking** - 作業時間の計測と記録
- 👥 **Team Management** - プロジェクトメンバーとタスクの管理
- 📊 **Work Report** - 工数と作業内容のレポート機能
